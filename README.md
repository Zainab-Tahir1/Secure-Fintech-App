Secure FinTech App

A security-aware FinTech mini-app built with Flask that demonstrates practical defenses:

✅ User registration & login (hashed passwords, login lockout)

✅ Strong password policy & client/server validation

✅ Session management & idle timeout

✅ Encrypted data at rest (Fernet)

✅ Input sanitization & safe error handling

✅ File upload validation (type/size, secure filenames)

✅ Audit/activity logs for traceability

✅ ≥ 20 manual cybersecurity tests documented for evaluation
